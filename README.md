A simple and responsive temperature conversion web app built for the Oasis Infobyte Internship Level 1 Task 3.

https://my-temperature-converter-example.netlify.app/

## Overview
This project enables users to convert temperatures between Celsius and Fahrenheit. Users enter a numeric temperature, choose the input unit, and click the convert button. The converted temperature is displayed immediately with the correct unit.

## Features
- Convert temperature from Celsius to Fahrenheit
- Convert temperature from Fahrenheit to Celsius
- Clear and user-friendly interface
- Responsive design for desktop and mobile browsers

## Project Structure
- `Task 3/OIBSIP-Level1-Task3/OIBSIP-Level1-Task3-Pratul/index.html` - Main HTML file
- `Task 3/OIBSIP-Level1-Task3/OIBSIP-Level1-Task3-Pratul/style.css` - Styling and layout
- `Task 3/OIBSIP-Level1-Task3/OIBSIP-Level1-Task3-Pratul/script.js` - Conversion logic and interaction handling

## Usage
1. Open the `index.html` file in a browser, or view the live demo.
2. Enter a temperature value.
3. Select the input unit: Celsius or Fahrenheit.
4. Click the "Convert" button.
5. View the converted result below the input.
